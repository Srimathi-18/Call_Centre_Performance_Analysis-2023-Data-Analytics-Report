# 📞 Call Centre Performance Analysis 2023 – Data Analytics Project

## 🧩 Overview
This project analyzes the 2023 Call Centre data to evaluate agent performance, call handling efficiency, and customer satisfaction trends. Using **Microsoft Excel**, a comprehensive report and dashboard were created to visualize key metrics and highlight improvement areas.

---

## 🎯 Objectives
- Examine call trends across different months and weekdays.
- Assess agent productivity and call response rates.
- Measure customer satisfaction and feedback performance.
- Build a visual, interactive Excel dashboard for management decisions.

---

## 🧰 Tools & Techniques
- **Microsoft Excel 2023**
  - Pivot Tables for data summarization.
  - Charts (Column, Line, and Pie) for visual representation.
  - Conditional Formatting for highlighting KPIs.
  - Slicers for dynamic filtering and interactivity.

---

## 📈 Insights
- 📞 Peak call volumes were observed during **Q2 (April–June)**.  
- ⚙️ **Average handling time** reduced by 15% compared to previous year.  
- ⭐ **Customer satisfaction** consistently remained above 90%.  
- 🧑‍💼 Identified top-performing agents contributing to faster resolutions.
